📚 Documentație Restaurant Management System

🚀 Pornire Rapidă

Sistemul este configurat cu 336 produse și 797 ingrediente gata de utilizare.

  • Explorează meniul: comanda.html
  • Personalizează: Catalog Produse
  • Populează stocuri: Financiar → NIR
📖 Ghid Complet ⚙️ Administrare

📦 Module Principale

  • Stocuri: Ingrediente, rețete, fișe tehnice
  • Financiar: NIR, rapoarte, comenzi
  • Marketing: Loialitate, feedback
  • Raportare: Dashboard, analytics

⚙️ Configurare

Opțiunea A - Template (Recomandat):

  1. Delete unused products (1-2 hours)
  2. Ajustează prețurile (1 oră)
  3. Populează stocurile prin NIR (30 min)

Timp total: 3-4 ore

🔄 Flux Zilnic

Dimineața: Verifică stocuri, adaugă ingrediente prin NIR

În timpul zilei: Comenzi → KDS → Stocuri scad automat

Seara: Raport Z, verifică anulări, backup

🆘 Troubleshooting

Server nu pornește:

Get-Process -Name node | Stop-Process -Force

Stocurile nu scad:

Verifică dacă produsele au rețete definite în Stocuri → Rețete

Auto-Restart Server:

Vezi fișierele AUTO-RESTART-GUIDE.md și watchdog.ps1
