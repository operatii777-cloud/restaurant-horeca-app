⚪ Test 1: Flux Comandă Completă ⭐⭐⭐ CRITIC

▼

📱 Pas 1: Client creează comandă

Acțiuni:

  1. Deschide tab-ul "Client - Comandă"
  2. Selectează Masa 5
  3. Adaugă 2 produse diferite din meniu
  4. Verifică că prețul total se calculează corect
  5. Apasă butonul "Trimite Comandă"

Rezultat așteptat: Mesaj de confirmare + coș se golește

🍽️ Pas 2: Ospătar primește comanda

Verificări:

  • Comanda apare INSTANT în tab-ul "Ospătar 1"?
  • Auzi sunet de notificare?
  • Toate produsele sunt listate corect?
  • Masa 5 este afișată?

👨‍🍳 Pas 3: Bucătărie procesează

Acțiuni:

  1. Comanda apare INSTANT în tab-ul "Bucătărie"?
  2. Click pe "Start Preparation"
  3. Status se schimbă în "In Preparation"?
  4. Ospătarul vede update-ul INSTANT (fără refresh)?
  5. Click pe "Ready"
  6. Status se schimbă în "Ready"?

✅ Pas 4: Ospătar finalizează

Acțiuni:

  1. Click pe "Delivered" în tab-ul Ospătar
  2. Status se schimbă în "Delivered"?
  3. Comanda dispare din listă sau merge în istoric?

📝 Notițe (dacă ai găsit probleme)

⚪ Test 2: Socket.IO & Notificări în Timp Real ⭐⭐⭐ CRITIC

▼

🔌 Verificare conexiune Socket.IO

Acțiuni:

  1. Deschide tab-ul "Admin Advanced"
  2. Verifică în colțul dreapta sus: 🟢 "Socket.IO Live"?
  3. Deschide consola browser (F12)
  4. Verifică mesaje de conectare Socket.IO

📡 Test actualizări în timp real

Acțiuni:

  1. Creează o comandă nouă din tab-ul Client
  2. Admin Advanced se actualizează INSTANT (fără refresh)?
  3. Numărul de comenzi crește automat?

🔄 Test reconectare

Acțiuni:

  1. În terminal, oprește serverul (Ctrl+C)
  2. Indicatorul devine 🔴 "Socket.IO Offline"?
  3. Pornește serverul din nou (npm run dev)
  4. Indicatorul devine 🟢 "Socket.IO Live" automat?
